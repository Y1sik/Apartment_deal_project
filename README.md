## 📖 프로젝트 소개

`Apartment_deal`은 아파트 매매검색 웹 홈페이지 입니다.



- 기획배경

  - 아파트 매매 관련 커뮤니티와 매매검색 주변 상권에 대한 정보를 포함한 웹페이지 설계.

  - 관심 매물 등록 페이지를 만들고자 함
